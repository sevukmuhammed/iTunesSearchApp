#  SearchApp - iOS

[![Swift Version][swift-image]][swift-url]


SearchApp is an iOS application developed using RxSwift



## Requirements

- iOS 11.0+

- Xcode 11.5+


## Architecture and Concepts

- MVVM

- Reactive Functional Programming (RxSwift, RxCocoa)

- Dependency Injection (Swinject)

- Coordinator Pattern


## Build Steps

run "pod install"
